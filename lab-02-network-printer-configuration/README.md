# Lab 02 — Network Printer Configuration

## Overview

This lab simulates the installation and configuration of a network printer in a small office environment.

The scenario focuses on preparing the network, assigning a stable IP address to the printer, testing connectivity, and documenting the installation for future support.

## Objectives

- Analyze the existing network before installing the printer
- Choose between DHCP reservation and static IP addressing
- Configure a stable IP address for a network printer
- Test connectivity from a user workstation
- Understand how the printer would be added in Windows
- Document the installation for future troubleshooting

## Technologies and Concepts

- Network printer configuration
- Static IP addressing
- DHCP planning
- Default gateway and DNS settings
- Ping connectivity testing
- Windows printer installation process
- Printer driver considerations
- Troubleshooting methodology
