# Lab 04 — CCTV / IP Camera Support

## Overview

This lab simulates a basic CCTV/IP camera support environment for a small store or office.

The scenario includes a local network with a router, switch, administrative computer, Server-NVR, and simulated IP cameras.

## Objectives

- Understand the main components of a CCTV/IP camera environment
- Configure static IP addresses for cameras and the NVR/server
- Use DHCP for the administrative computer
- Simulate access to an NVR/CCTV management interface
- Validate connectivity between the PC-Admin, gateway, cameras, and Server-NVR
- Document CCTV troubleshooting steps

## Technologies and Concepts

- Cisco Packet Tracer
- CCTV/IP camera support
- Static IP addressing
- DHCP
- Server/NVR concept
- HTTP service simulation
- Ping connectivity testing
- Network troubleshooting
- Technical documentation

This is an educational lab created for IT support and infrastructure practice. Packet Tracer does not simulate real camera video, so the focus is on network design, IP configuration, connectivity validation, and CCTV support workflow.
