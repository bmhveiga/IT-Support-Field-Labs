# Lab 03 — POS Support Lab

## Overview

This lab simulates a basic POS support environment for a small store or office.

The scenario includes a POS workstation, an administrative computer, a receipt printer, and a POS server/application accessible over the local network.

## Objectives

- Understand the main components of a POS environment
- Configure a small local network for POS support
- Use DHCP for workstation addressing
- Configure static IP addresses for critical devices
- Simulate POS application access through a local server
- Validate connectivity between the POS workstation, gateway, receipt printer, and server
- Document common POS peripherals and troubleshooting steps

## Technologies and Concepts

- Cisco Packet Tracer
- POS workstation support
- Receipt printer support
- DHCP
- Static IP addressing
- HTTP service simulation
- Ping connectivity testing
- Peripheral troubleshooting
- Small business IT support
